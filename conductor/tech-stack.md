# Tech Stack

## Architecture
Offline-First Desktop App where both the Backend and Frontend run locally on the cashier's machine.

## Components
- **Backend (Server):** Python 3.11+, Django, Django Rest Framework (DRF), SQLite/PostgreSQL.
- **Frontend (Desktop/Web):** React 19, Modern Functional Components, Vite/Next.js.
- **Mobile App:** Flutter.
- **Testing:** Comprehensive E2E testing suite.
- **AI Agent Framework:** Google Antigravity 2.0 Multi-Agent Setup.
