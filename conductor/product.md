# Product Context

**VentaPOS NextGen**
A high-performance, offline-first Point of Sale (POS) system for retail and wholesale businesses.

## Target Users
- Cashiers, Store Managers, and Admins operating in dynamic retail environments.

## Core Features
- Full Offline-First capability (Backend and Frontend run locally).
- Sales, Inventory, and Ledger tracking.
- Multi-branch and tenant synchronization.
- Comprehensive Reporting.
- Fully localized in Egyptian Arabic.

## Goals
- Complete all app pages across the system.
- Ensure flawless E2E testing across all scenarios.
